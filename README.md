# Learn Programming

A collection of programming learning materials organized as markdown files.

## Content Structure

Browse the `/content` folder to explore different topics:

- **00-overview** - Introduction and overview
- **01-type-systems** - Type theory, advanced types, type inference
- **02-oop-principles** - SOLID principles, composition, encapsulation, polymorphism
- **03-design-patterns** - Creational, structural, and behavioral patterns
- **04-functional-programming** - Pure functions, immutability, monads, functors
- **05-async-programming** - Event loop, promises, async/await, concurrency
- **06-error-handling** - Exceptions, result types, error propagation
- **07-data-structures** - Linear, trees, hash tables, graphs
- **08-architecture** - Layered, hexagonal, microservices, clean architecture
- **09-testing** - Unit testing, TDD, mocking, integration testing
- **10-performance** - Profiling, algorithms, memory, rendering optimization
- **11-ai-era-programming** - AI capabilities, prompts, and modern development

## Viewing

Browse these files directly on GitHub or clone the repository and view them in your favorite markdown editor.
